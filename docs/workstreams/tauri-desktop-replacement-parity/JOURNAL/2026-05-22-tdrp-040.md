# 2026-05-22 — TDRP-040 packaged sidecar and Windows NSIS build

## Scope

- Implement deterministic packaged sidecar strategy for the Tauri desktop app.
- Produce a Windows installer artifact without requiring `CODEX_HELPER_CLI_PATH`.
- Avoid touching the developer machine's active codex-helper runtime during live smoke.

## Changes

- Added `apps/desktop/scripts/prepare-sidecar.mjs`.
- Added `apps/desktop/src-tauri/sidecars/.gitignore`.
- Enabled Tauri bundling with Windows NSIS target and `bundle.externalBin`.
- Changed desktop proxy start to resolve packaged resource sidecar before sibling dev binary and env fallback.
- Added control-command tests for sidecar resolution order.
- Added `docs/DESKTOP_RELEASE.md`.
- Updated README/README_EN/CHANGELOG and workstream evidence.
