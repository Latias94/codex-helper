# TDRP-020 Path Actions And Lightweight Config Import/Export

Date: 2026-05-22

## Summary

- Added real Tauri path actions for home/config/logs/cache.
- Added lightweight single-config export/import commands for `config.toml`.
- Import validates supported TOML config shapes, creates a timestamped backup before replacement, and leaves the current config untouched on validation failure.
- Wired Settings buttons through `tauri-plugin-dialog` and `tauri-plugin-opener`.
- Added warning copy that exported TOML may contain inline secrets.
