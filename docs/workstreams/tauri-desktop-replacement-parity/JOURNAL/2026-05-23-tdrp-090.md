# 2026-05-23 — TDRP-090 replacement docs and egui deprecation

## Scope

- Update user-facing docs after the Windows packaged Tauri lifecycle gate passed.
- Decide whether to remove or deprecate `codex-helper-gui`/egui.
- Keep the replacement claim aligned with actual evidence.

## Decision

- Deprecated egui instead of removing it.
- Rationale:
  - Windows NSIS packaged Tauri smoke is green.
  - macOS/Linux packaged parity has not been smoked.
  - The old egui binary is still useful as rollback/non-Windows fallback while release operations mature.

## Changes

- README and README_EN now describe Tauri as the verified Windows packaged GUI replacement path.
- CHANGELOG Unreleased no longer frames Tauri only as source-preview/internal dogfood.
- docs/DESKTOP_RELEASE moves full packaged lifecycle smoke into the Windows verified list.
- `src/bin/codex-helper-gui.rs` prints a deprecation warning when launched from a visible console.
- Workstream TODO/HANDOFF/MILESTONES/WORKSTREAM/EVIDENCE records TDRP-090 completion and hands off to TDRP-100.

## Verification

- `cargo check --features gui --bin codex-helper-gui`: PASS.
- `python -m json.tool docs\workstreams\tauri-desktop-replacement-parity\WORKSTREAM.json`: PASS.

## Result

- TDRP-090 is DONE_WITH_CONCERNS.
